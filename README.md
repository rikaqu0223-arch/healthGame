# BioCurrent

A hackathon MVP for a stylized Three.js health game. Pilot a tiny bio-explorer submarine through a cardiovascular tunnel, collect light crystals, avoid cholesterol blocks, and use sonar pulses to restore flow.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech

- Vite
- Three.js
- Vanilla JavaScript
